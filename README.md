# Video Game Sales Analysis

This project analyzes best-selling video game sales data using Python fundamentals including variables, lists, loops, conditionals, dictionaries, and functions.

## Dataset Source
Data adapted from VGChartz (public domain estimates) containing the top 20 best-selling video games of all time with information on rank, name, platform, release year, genre, publisher, and regional sales (North America, Europe, Japan, and global).

## Key Analyses Performed
- Calculated total games, average global sales, and top game market share
- Cleaned and formatted string data (whitespace removal, case conversion)
- Filtered games by sales thresholds and release years
- Aggregated sales by genre and publisher using dictionaries
- Created reusable functions for sales calculation, genre filtering, and game summaries

## Technologies Used
- Python 3
- Built-in data structures (lists, tuples, dictionaries)
- Loops and conditional statements
- String methods and f-strings
- Functions with default parameters
